## Causal Discovery & Structure Learning

### FoundCause: Causal Discovery with Latent Confounders from Observational Data
**Patrick Blöbaum, Krishnakumar Balasubramanian, Shiva Prasad Kasiviswanathan**
[arXiv:2606.17516v1](https://arxiv.org/abs/2606.17516v1) | `cs.LG`, `cs.AI`, `stat.ME`, `stat.ML`

Proposes FoundCause, an amortized causal discovery model trained on synthetic data that maps datasets directly to causal graphs (including latent confounders) in a single forward pass, without requiring interventions.

*Why relevant:* Directly addresses causal discovery under latent confounding from observational data -- a core challenge for structure learning in causally insufficient systems.

### Tensor-based second-order causal discovery
**Nathan Ouyang, Kexin Wan, Anna Seigal**
[arXiv:2606.18074v1](https://arxiv.org/abs/2606.18074v1) | `stat.ML`, `cs.LG`, `stat.ME`

Introduces TSCD, a tensor-based algorithm that leverages covariance matrices from observational and interventional data for causal discovery in linear SEMs, requiring only uncorrelated noise.

*Why relevant:* Novel algebraic approach to interventional causal discovery; connects tensor decompositions with DAG identifiability from mixed observational-interventional data.

### Causal invariance in graphical models with latent variables
**Marco Borriero, Monia Lupparelli, Giovanni M. Marchetti, Veronica Vinciotti**
[arXiv:2606.13281v1](https://arxiv.org/abs/2606.13281v1) | `stat.ME`

Studies the causal invariance principle for discovering causal parents when some parents are unobserved. Shows that the induced graph over observed variables may no longer be a DAG, and develops methods for identification using Markov equivalence under latent variables.

*Why relevant:* Core topic: causal invariance + latent variables + graphical model equivalence classes -- directly connects invariant prediction with ancestral graph theory.

### Constraint-based difference graph discovery in a linear setting
**Daria Bystrova, Emilie Devijver**
[arXiv:2606.13947v1](https://arxiv.org/abs/2606.13947v1) | `stat.ME`

Proposes a constraint-based method for inferring difference graphs between two environments under linear SCMs. Introduces "diff-separation," a new graphical criterion extending d-separation to characterize invariance of regression coefficients across environments.

*Why relevant:* Extends constraint-based structure learning to multi-environment settings; the diff-separation criterion is a novel graphical tool for comparing causal structures.

## Conditional Independence Testing

### Fast Nonparametric Conditional Independence Testing via Two-Stage Regression
**Eric V. Strobl**
[arXiv:2606.18011v1](https://arxiv.org/abs/2606.18011v1) | `stat.ML`, `cs.LG`, `stat.ME`

Introduces BLITZ, a fast nonparametric conditional independence test based on two-stage residualization. Under a generalized additive model, the test statistic reduces to a simple residual correlation while remaining semiparametrically efficient, with validity guaranteed via sample splitting and cross-fitting.

*Why relevant:* Fast CI testing is a core primitive for constraint-based causal discovery; the semiparametric efficiency result under nonlinear nuisance estimation is noteworthy.

[arXiv:2606.12680v1](https://arxiv.org/abs/2606.12680v1) | `cs.LG`, `stat.ML`
