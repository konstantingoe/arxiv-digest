## Conditional Independence Testing

### Fast Nonparametric Conditional Independence Testing via Two-Stage Regression
[arXiv:2606.18011](https://arxiv.org/abs/2606.18011) | `stat.ML`, `stat.ME`

Proposes a computationally efficient nonparametric CI test using a two-stage regression approach. Under a generalized additive nonparametric model, the semiparametrically efficient test statistic reduces to a simple residual correlation, which remains asymptotically valid when nuisance functions are estimated with flexible ML methods via sample splitting and cross-fitting.

*Why relevant:* Fast, nonparametric CI testing is a core primitive for constraint-based causal discovery; the semiparametric efficiency result is noteworthy for high-dimensional applications.

## Semiparametric & Nonparametric Causal Methods

### Prediction-Powered Causal Inference by Automatic Debiased Machine Learning and Semi-Supervised Riesz Regression
**Masahiro Kato**
[arXiv:2606.12892](https://arxiv.org/abs/2606.12892) | `stat.ML`, `stat.ME`

Derives the semiparametric efficiency bound for causal/structural parameters when unlabeled auxiliary regressors are available alongside labeled data. Shows that leveraging unlabeled data can strictly reduce asymptotic variance below the labeled-only bound. Combines efficient influence functions with DML to propose EE-DML-PPCI and TMLE-DML-PPCI estimators.

*Why relevant:* Semiparametric efficiency theory for causal inference; relevant to high-dimensional settings where auxiliary data can improve estimation of nuisance functions.

### Causal Sufficient Dimension Reduction for Multiple Continuous Exposures with an Application to Environmental Mixtures
**Thomas W. Hsiao, Howard H. Chang, Razieh Nabi**
[arXiv:2606.14840](https://arxiv.org/abs/2606.14840) | `stat.ME`

Develops a semiparametric framework (CSDR) for representing causal exposure-response surfaces through low-dimensional exposure summaries. *Why relevant:* Semiparametric causal inference for multivariate treatments; connects dimension reduction with causal identification under potential outcomes.
